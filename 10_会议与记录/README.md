## 会议体系

### 每日晨会（15分钟）
- **时间**：每天上班前15分钟
- **参与人**：生产主管、班组长、工艺、质量
- **议程**：
  1. 昨日产量、质量、安全通报（3分钟）
  2. 今日重点任务和计划（5分钟）
  3. 昨日问题反馈和当日风险预警（5分钟）
  4. 同步信息（2分钟）
- **输出**：晨会记录

### 周会（60分钟）
- **时间**：每周一/五下午
- **参与人**：部门负责人及以上
- **议程**：
  1. 本周KPI达成情况（15分钟）
  2. 重点项目进度（15分钟）
  3. 问题协调与资源需求（15分钟）
  4. 下周重点计划（15分钟）
- **输出**：周会纪要 + 待办追踪

### 月度经营会（2小时）
- **时间**：次月第5个工作日前
- **参与人**：全体管理层
- **议程**：
  1. 上月经营指标回顾（OEE、良率、交付、成本）
  2. 改善项目成果汇报
  3. 偏差分析与对策
  4. 下月目标与资源配置
  5. 战略专题讨论（季度）
- **输出**：月度经营报告 + 改善行动计划

## 会议管理原则

- **会前有议程**：提前发送会议议程和资料
- **会中有纪要**：当场记录决议和待办
- **会后有追踪**：待办事项明确负责人和截止日期
- **问题有闭环**：上次会议待办须更新进展

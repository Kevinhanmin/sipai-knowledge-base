# ⏱️ SMED — 快速换模

> Single Minute Exchange of Die — 单分钟换模

---

## 核心理念

把所有换型操作分为**内部作业**（停机做）和**外部作业**（运行中做），然后通过将内部转化为外部来大幅缩短停机时间。

## SMED四大步骤

### Step 1: 观察记录当前换型
- 用视频录制完整换型过程
- 把每个动作记录下来，区分：内部作业 / 外部作业

### Step 2: 区分内外作业
| 类别 | 定义 | 示例 |
|------|------|------|
| 内部作业（IED） | 必须停机才能进行的操作 | 拆装模具、调整参数 |
| 外部作业（OED） | 可以在设备运行时进行 | 准备工具、预热模具、搬运物料 |

### Step 3: 内部作业→外部作业
这是SMED的核心。通过以下手段将内部转化：

| 手段 | 说明 | 案例 |
|------|------|------|
| 快速夹紧 | 螺栓→快夹 | 液压夹钳替代螺丝 |
| 定位标准化 | 消除调校 | 定位块、导向槽 |
| 预调准备 | 外部提前做好 | 模具预热、参数预设 |
| 取消调整 | 采用一次性定位 | 自动停止定位 |
| 并行作业 | 多人同时操作 | 两人配合换模 |

### Step 4: 优化所有操作
- 内外作业全面优化
- 规范化操作流程
- 可视化操作指导
- 工具标准化、定置化

## SMED改进案例

| 阶段 | 换型时间 | 主要措施 |
|------|---------|---------|
| 改善前 | 60分钟 | 拧螺栓、调参数、找工具 |
| 区分内外 | 40分钟 | 提前备模→减少停机37% |
| 内转外 | 20分钟 | 快夹+定位块→减少50% |
| 全面优化 | 9分钟 | 并行+标准化→减少55% |

## SMED检查清单

### 准备阶段
- [ ] 模具/工装是否提前准备到位？
- [ ] 工具、量具、辅具是否齐全？
- [ ] 下一批物料是否到位？
- [ ] 操作人员是否已就位？

### 执行阶段
- [ ] 是不是全部外部作业在设备运行中做了？
- [ ] 有没有动作可以合并或省略？
- [ ] 有没有可以用快夹代替螺栓的地方？
- [ ] 有没有可以通过标识减少调整的步骤？

### 复盘阶段
- [ ] 实际换型时间记录了吗？
- [ ] 遇到什么问题？如何解决？
- [ ] SMED工作文档是否更新？

## SMED常见误区

| 误区 | 纠正 |
|------|------|
| 只有模具大才需要SMED | 任何换型/切换都适用 |
| SMED只看时间，不看质量 | 不能因为快而牺牲首件质量 |
| 一次性做到十分钟内 | 逐步优化，每次进步20-30% |
| 不改造设备就没法做 | 90%靠管理改善，10%靠设备改造 |

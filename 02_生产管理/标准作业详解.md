# 📋 标准作业

> 标准化是改善的基石 — 没有标准就没有改善

---

## 标准作业三要素

### 1. 节拍时间（Takt Time）
```
节拍时间 = 可用工作时间 / 客户需求量
```
> 例：460分钟/班 ÷ 460件/班 = 1分钟/件

### 2. 作业顺序
- 操作工在一个循环内执行的操作步骤
- 按最优排列，减少移动和等待
- 步骤必须编号、明确起止点

### 3. 标准手持（Standard WIP）
- 维持流程顺畅所需的最小在制品
- 包括：设备上的、工位间的、工序间的
- 过多=浪费，过少=断料风险

## 标准作业三大文件

### 1. 工序能力表（Process Capacity Sheet）
- 记录每个工序的：加工时间、换型时间、设备台数
- 用于计算每个工序的产能瓶颈

### 2. 标准作业组合表（Standard Work Combination Table）
- 人机交互时间图
- 区分：手工作业时间、步行时间、设备加工时间

### 3. 标准作业指导书（Standard Work Instruction Sheet）
- 操作位置布局图
- 标注作业步骤、品质检查点、安全注意点

## SOP编写实战指南

### 好的SOP标准
1. **小白能看懂** — 没有行业经验的人看了也能操作
2. **操作步骤精确** — 每一步只做一件事
3. **关键点明确** — 标出"这里错了会出大问题"
4. **图文并茂** — 一张好图胜过100字
5. **版本受控** — 由审批、变更、废止流程

### SOP编写流程

```
确认工序 → 观察录制 → 步骤分解 → 标注要点 → 配图拍照 → 评审修改 → 现场验证 → 批准发布 → 培训执行 → 定期复核
```

## 标准作业的持续改善

- **遵循标准**：按标准作业执行
- **发现问题**：比标准更好/更差都是改善机会
- **改善标准**：验证新方法后修订标准
- **培训新标准**：全员培训后才算标准生效

> 没有标准就没有改善！
> 有了标准但不遵守，就是管理失职！

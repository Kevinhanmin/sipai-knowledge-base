# 📱 电子行业案例库

> 包括SMT、组装、测试全流程精益数字化案例

---

## 行业特征

| 特征 | 说明 |
|------|------|
| 产品生命周期短 | 新品迭代快，需要快速换线和柔性排产 |
| 质量要求极高 | 返修成本高，一次良率至关重要 |
| 自动化程度高 | SMT线基本全自动，组装段预自动化趋势 |
| 用工密集 | 组装、测试、包装环节仍大量依赖人工 |
| 物料种类多 | BOM层级多，物料替代管理复杂 |

## 电子行业关键精益点

### SMT车间
- **换线提速**：SMED——双轨换线、离线编程、喂料器预装
- **抛料率管控**：<0.3%为优秀，每周统计改善
- **炉温监控**：SPC管控回流焊温度曲线，CPK≥1.33
- **物料防错**：扫码防错、料盘上机前校验

### 组装段
- **防错设计**：治具防呆、传感器检测、视觉定位
- **工位平衡**：线平衡率目标≥85%，瓶颈工位优先改善
- **静电防护**：ESD全程管控，月定期稽核

### 测试段
- **一次测试通过率（FPY）**目标>95%
- **测试站OEE**：减少等待和程序加载时间
- **数据分析**：不良品关联分析找根本原因

## 适用场景速查

| 问题类型 | 推荐工具 | 预期效果 |
|---------|---------|---------|
| SMT换线时间长 | SMED+离线编程 | 缩短50%+ |
| 组装线不平衡 | 山积图+工位重排 | 效率提升20%+ |
| 测试一次通过率低 | 分层FMEA+DOE | FPY提升至95%+ |
| 物料错料率高 | 扫码防错系统 | 错料降至0 |
| 产品换型频繁 | 快速换线+柔性排产 | 换线时间<15分钟 |

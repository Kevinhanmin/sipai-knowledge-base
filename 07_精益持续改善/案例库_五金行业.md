# 🔩 五金行业案例库

> 包括冲压、机加工、钣金、压铸等案例

---

## 行业特征

| 特征 | 说明 |
|------|------|
| 模具为核心资产 | 模具对效率和质量影响巨大 |
| 批量化生产 | 大批量→模具寿命、小批量→换型频繁 |
| 设备密集型 | OEE管理价值高，停机损失大 |
| 材料成本占比高 | 材料利用率直接影响利润率 |
| 质检依赖经验 | 尺寸/外观检测自动化程度低 |

## 五金行业关键精益点

### 冲压车间
- **模具管理**：模具寿命台账、预防性维护、快换系统
- **材料利用率**：卷材利用率（目标>95%）、废料回收
- **冲压节拍**：SPM（冲次/分钟）优化，减少空冲
- **首件检验**：首件快速检测流程（<5分钟出结果）

### 机加工
- **CNC利用率**：主轴运转率（目标>85%）
- **刀具管理**：刀具寿命管理、对刀仪预调
- **换型提速**：SMED——工装快换、程序预加载
- **冷却液管理**：浓度监控、过滤再生、寿命台账

### 钣金
- **排版优化**：Nest软件优化排版（利用率目标>85%）
- **激光/冲床共用**：复合加工减少流转
- **焊接质量**：焊接参数SPC监控

# 💉 注塑行业案例库

> 包括注塑成型、模具、辅机等精益案例

---

## 行业特征

| 特征 | 说明 |
|------|------|
| 工艺参数敏感 | 温度/压力/速度/时间多变量耦合 |
| 模具投入高 | 一副模具几十上百万，急需提升利用率 |
| 材料成本波动大 | 塑料原料价格波动影响成本控制 |
| 不良品难避免 | 缩水、飞边、困气、变形等常见不良 |
| 机台种类多 | 吨位、品牌、型号差异化大，排产复杂 |

## 注塑行业关键精益点

### 工艺优化
- **成型参数标准化**：设定标准工艺窗口，减少试模次数
- **模温控制**：模温机精度管理，波动<±1°C
- **科学注塑（Scientific Molding）**：型腔压力曲线监控
- **DOE参数优化**：关键尺寸的CPK≥1.33

### 模具管理
- **模具保养**：周期性保养+状态评估（建议每10万模次保养）
- **模具寿命台账**：记录累计冲次、维修记录、状态等级
- **模具快换**：永磁吸盘/液压快换+水温预热

### 生产管理
- **机台排产**：考虑材料颜色切换（浅→深为佳）、吨位匹配
- **机台OEE**：重点关注注塑周期是否达到理论值
- **集中供料**：中央供料系统减少人工和误判
- **机械手自动化**：取出+水口剪切+摆放一体化

### 不良率管控（行业参考）

| 不良类型 | 可接受水平 | 改善方向 |
|---------|-----------|---------|
| 外观不良（飞边/缩水）| <1% | 工艺优化+模具维护 |
| 尺寸不合格 | <0.5% | SPC过程控制 |
| 材料混料 | 0 | 防错+条码追溯 |
| 短射/困气 | <0.3% | 模具排气改善 |

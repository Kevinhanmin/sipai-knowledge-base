## 工艺管理体系

### 文件控制
- **工艺文件编制流程**：设计→评审→批准→发布→培训→执行
- **版本管理**：所有工艺文件需标注版本号，变更需走ECN流程
- **工艺纪律检查**：每月至少一次现场工艺执行合规检查

### PFMEA方法论
```mermaid
graph LR
    A[识别工序] --> B[分析潜在失效]
    B --> C[评估严重度/频度/探测度]
    C --> D[RPN优先级排序]
    D --> E[制定改善措施]
    E --> F[重新评估RPN]
    F --> G[标准化]
    G --> A
```

### SOP编写原则
- **图文并茂**：关键步骤配示意图或照片
- **防错设计**：SOP中需标注防错点和关键控制点
- **作业验证**：新SOP需经老员工和白纸员工双向验证
- **可视化**：作业要点用颜色/图标区分重点

## 关键工艺控制点

| 工序类型 | 常见关键参数 | 管控手段 |
|---------|------------|---------|
| 注塑 | 温度、压力、保压时间 | SPC控制图 |
| 冲压 | 冲速、模具间隙、材料厚度 | 首末件检验 |
| 焊接 | 电流、电压、焊速 | 破坏性试验 |
| SMT贴片 | 回流焊温度曲线 | 炉温测试板 |
| 装配 | 扭矩、压入力 | 扭矩校验仪 |

# 📊 诊断报告模板

> LSIF免费诊断报告HTML模板说明

---

## 报告模板结构

### 页眉区域
- 思派工业Logo
- 报告标题：「精益智能工厂诊断报告」
- 客户企业名 + 报告编号 + 生成日期

### 第一部分：综合评级
- **评级卡片**（标杆/优秀/标准/改善/初始）
- 总分（满分5.0分）
- 一句话总结（AI生成）

### 第二部分：8维度雷达图
- 8轴雷达图对比
- 标注各维度得分
- 标注行业平均线（可选）

### 第三部分：维度评分详情
每个维度列出：
- 维度名称与图标
- 得分（1-5分）
- 简短评价（AI生成）

### 第四部分：评分曲线
- 所有题目得分折线图
- 标注高分项（优势）和低分项（改善点）

### 第五部分：改善建议
按优先级排序：
- 🔴 紧急（立即行动）
- 🟡 重要（短期计划）
- 🟢 建议（中长期规划）

### 页脚
- 思派工业联系方式
- 公众号/视频号二维码
- 「免费诊断仅限前100家」等营销话术

## HTML模板CSS规范

- 主色：#2563eb（蓝色）
- 辅色：#10b981（绿色，优秀项）
- 警示色：#ef4444（红色，需改进项）
- 字体：无衬线字体（中文用系统默认）
- 圆角：8px
- 最大宽度：800px，居中

## 评分脚本接口

```python
# 报告生成函数原型
def generate_report(
    company_name: str,
    scores: dict[str, float],   # {"战略与领导力": 3.5, ...}
    total_score: float,
    grade: str,
    suggestions: list[str],
    output_path: str
) -> str:  # 返回HTML文件路径
```

## 字段映射

| 飞书字段 | 报告字段 | 类型 |
|---------|---------|------|
| 公司名称 | company_name | str |
| 各维度得分×8 | scores | dict |
| 综合得分 | total_score | float |
| 评级 | grade | str |
| 改善建议 | suggestions | list[str] |
| 报告时间 | report_date | str |

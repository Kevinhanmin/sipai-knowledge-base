## 问卷评分公式

```
维度得分 = Σ(该维度各题得分) / 该维度总题数 × 维度权重

综合得分 = Σ(各维度得分)

评级：
  4.0-5.0 = 标杆级
  3.0-3.9 = 优秀级
  2.0-2.9 = 标准级
  1.0-1.9 = 改善级
  <1.0   = 初始级
```

## 供应链管理理念

### 精益供应链核心原则
1. **拉动式生产** — 按客户需求拉动，不生产多余库存
2. **JIT准时化** — 物料在需要的时间到达需要的位置
3. **看板管理** — 用看板信号驱动物料流动
4. **供应商协同** — 将供应商纳入精益体系
5. **物流优化** — 路线规划、容器标准化、水蜘蛛配送

### 库存管理关键指标

| 指标 | 公式 | 行业标杆 | 说明 |
|-----|------|---------|------|
| 库存周转率 | COGS / 平均库存 | 汽车行业8-12次/年 | 越高越好 |
| 呆滞料占比 | 呆滞金额/总库存 | <3% | 需定期处理 |
| 库存准确率 | 盘点一致项/总项 | >99% | 循环盘点保证 |
| 缺料频次 | 月缺料次数 | 0 | 确保生产效率 |
| 供应商准时交付率 | 按时交付批/总批 | >98% | 到货管理 |

### ABC分类法

| 分类 | 占比(金额) | 占比(项数) | 管理策略 |
|------|-----------|-----------|---------|
| A类 | 70-80% | 10-20% | 严格管控、定期盘点(JIT) |
| B类 | 15-20% | 20-30% | 适当管控、定量采购 |
| C类 | 5-10% | 50-70% | 简化管理、双堆法 |

## 仓储管理5S标准

- **日清日结**：当天出入库当天入账
- **定置定位**：所有物料有固定位置、标识清晰
- **先进先出**：色标管理或系统管控
- **账卡物一致**：每日抽盘，月度全盘

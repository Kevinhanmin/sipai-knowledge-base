## 人才培养体系

### 培训四层次
```
┌─────────────────────────────────┐
│   Level 4: 改善能手 → 内部讲师  │
├─────────────────────────────────┤
│   Level 3: 多能工 → 全能工      │
├─────────────────────────────────┤
│   Level 2: 岗位技能认证 → 熟练工 │
├─────────────────────────────────┤
│   Level 1: 新员工入职培训 → 上岗 │
└─────────────────────────────────┘
```

### 技能矩阵管理
- **技能分类**：操作技能、质量技能、设备维护、安全技能
- **等级定义**：
  - ★☆☆ 入门——可监督下操作
  - ★★☆ 熟练——独立操作
  - ★★★ 精通——可培训他人
- **多能工目标**：关键工序至少3人可独立操作

### OJT岗位培训四步法
1. **说给他听** — 讲原理、讲标准、讲要点
2. **做给他看** — 示范操作，边做边讲
3. **让他做做看** — 新员工操作，指导者观察纠正
4. **检查确认** — 定期抽查，持续辅导直到独立

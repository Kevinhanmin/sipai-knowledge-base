## 系统架构概览

```mermaid
graph TB
    subgraph 现场层
        S[传感器/PLC]
        R[机器人]
        C[摄像头]
    end
    subgraph 控制层
        SCADA
        PLC_P[PLC程序]
    end
    subgraph 执行层
        MES
        WMS[仓储管理]
    end
    subgraph 分析层
        AI_QC[AI质检]
        AI_PM[预测性维护]
        BI[数据分析]
        DT[数字孪生]
    end
    subgraph 决策层
        AI_PS[AI排产]
        AI_LLM[工业大模型]
    end
    S --> SCADA
    C --> SCADA
    SCADA --> MES
    MES --> AI_QC
    MES --> AI_PM
    MES --> BI
    MES --> DT
    R --> MES
    BI --> AI_PS
    DT --> AI_PS
    AI_QC --> AI_LLM
    AI_PM --> AI_LLM
```
